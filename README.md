# EWFTS - Ephemeral Wrapper For Temporary Scripts
A Python-based easy to use wrapper that executes a file then deletes a file sepcified by -file or the last argument
for more info read the mandoc [Here](src/ewfts-arch/ewfts.1)
# Availabillity
Ships as .deb, .deb (termux), .rpm and .pkg.tar.xz
